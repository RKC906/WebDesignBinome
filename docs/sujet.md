Mini-projet — Web design (Mars 2026)

1. Objectifs
- Créer un site d'informations (FrontOffice) sur la guerre en Iran
- Créer un backoffice (BackOffice) pour gérer les contenus

2. Travail à réaliser
1) Créer le site FrontOffice pour afficher les contenus
2) Créer le site BackOffice pour gérer les contenus (CRUD articles/catégories/utilisateurs)
3) Concevoir et peupler la base de données
4) Optimiser le référencement (SEO)

3. Points à vérifier
- URL normalisées (réécriture / rewriting)
- Structure sémantique des pages (utilisation correcte de h1..h6)
- Utilisation pertinente des titres de page
- Balises meta (title, description, etc.)
- Attributs alt pour les images
- Tests Lighthouse locaux (mobile et desktop)

4. Techniques recommandées
- Langage : PHP (ou autre au choix)
- Base : MySQL (ou autre au choix)
- URL rewriting activé (mod_rewrite / équivalent)

5. Livrables / préparation pour livraison
- Archive ZIP contenant le site fonctionnel (conteneurs Docker)
- Dépôt en ligne public (GitHub ou GitLab) — fournir le lien
- Document technique incluant :
	- captures d'écran FO et BO
	- modélisation de la base de données
	- accès BO avec utilisateur/mot de passe par défaut
	- numéro ETU

6. Date limite
- Mardi 31 mars à 14:00
